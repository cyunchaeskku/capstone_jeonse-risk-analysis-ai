# 연구보고서

이 폴더는 주(week) 단위로 개발 연구 내용을 기록하는 공간이다.

## 작성 원칙

- 세부 구현 코드 설명보다, 작업하면서 마주친 문제와 해결 과정을 중심으로 적는다.
- 기록 항목은 다음을 우선한다.
  - 어떤 문제를 해결하려 했는가
  - 어떤 데이터나 외부 자원을 사용했는가
  - 어떤 어려움이 있었는가
  - 어떻게 해결했는가
  - 아직 남은 리스크는 무엇인가
- 문서는 주차별로 분리한다.
- 파일명은 `week-01.md`, `week-02.md` 형식으로 유지한다.

## 권장 섹션

- 목표
- 사용 데이터 / API
- 진행 내용
- 이슈와 해결
- 결과
- 남은 과제
